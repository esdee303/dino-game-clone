# Simple Phaser3 with TS Template

## Setup

1. Install Node.js [install it from here](https://nodejs.org/en/) (Node.js version >= 14.6.0 required)

2. Clone or download this repository

3. Navigate into the project directory

4. Install the requirements

```bash
  $ npm install
```

5. Run the app

```bash
  $ npm run dev
```
